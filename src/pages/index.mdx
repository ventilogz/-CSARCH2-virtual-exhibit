## Exhibit Overview

This virtual exhibit focuses on the evolution of Human-Computer Interaction or HCI. It shows how people interacted with computers from the early years of punch cards and batch processing up to modern interfaces like touchscreens, voice assistants, and spatial computing.

The exhibit explains how each milestone made computers easier, faster, and more user-friendly. It also shows how old computing methods helped shape the interfaces that we use today.

## Timeline of HCI Evolution

### 1940s–1950s: Batch Processing and Punch Cards

During the early years of computing, users interacted with computers through punch cards and batch processing. Instructions were prepared beforehand and processed in groups. This made interaction slow because users could not directly communicate with the computer in real time.

### 1960s–1970s: Command Line Interfaces

Command Line Interfaces allowed users to type text-based commands using keyboards and terminals. This was more interactive compared to punch cards because users could directly enter commands and receive responses from the computer.

### 1968: Pointing Devices

The computer mouse became an important milestone in HCI because it allowed users to control and select items on a screen using physical movement. This helped make computers easier to navigate and supported the development of graphical interfaces.

### 1980s: Graphical User Interfaces

Graphical User Interfaces introduced windows, icons, menus, and pointers. Instead of typing commands, users could click and interact with visual elements. This made computers easier for non-technical users.

### 2000s: Touch and Mobile Interfaces

Touchscreens and mobile devices changed how people interacted with technology. Users could tap, swipe, pinch, and zoom directly on the screen, making interaction more natural and intuitive.

### Present and Future: Spatial and Voice Computing

Modern HCI now includes voice assistants, augmented reality, virtual reality, and spatial computing. These technologies allow users to interact with computers through speech, gestures, and immersive digital environments.

## Interactive Elements

Our exhibit will include interactive elements such as an interactive timeline, punch card simulator, CLI terminal simulator, mini desktop simulator, and voice command simulation. These will help users understand the evolution of HCI in a more engaging way.
